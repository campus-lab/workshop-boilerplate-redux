# workshop-boilerplate-redux

A simple React+Redux workshop exercice.

[Slides](https://slides.com/guilhermecabral/bmw_v2/)

## HOW TO 

```
git clone https://github.com/campus-lab/workshop-boilerplate-redux.git
npm install
gulp watch
```

## DEPENDENCIES

+ NodeJS
    + https://nodejs.org/en/
+ Gulp
    + ```npm install -g gulp```
