# Workshop-boilerplate

A simple starter boilerplate for a React+Redux App.

Slides [here](https://slides.com/guilhermecabral/bmw_v2/live#/).

## HOW TO 

```
git clone https://github.com/campus-lab/workshop-boilerplate-redux.git
npm install
gulp watch
```

## Prerequisites

+ NodeJS
    + https://nodejs.org/en/
+ Gulp
    + ```npm install -g gulp```
